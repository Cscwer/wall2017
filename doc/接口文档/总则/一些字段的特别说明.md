# 一些字段的特别说明

## \_id 
在接口返回的数据中，请无视 `__v` 字段，这是 Mongoose 自己加上去的，无须理会。 

MongoDB 中每一条数据实例（专业术语是文档 document，可以看作是 SQL 的行），都会有一个 字段 `_id` 它的类型是 `ObjectId` 可以转为字符串，而且绝对唯一，不会重复，用作数据库索引，今后如果文档中提到 `id` 这样的字眼一般指的就是这个数据库自动创建的 `_id`
