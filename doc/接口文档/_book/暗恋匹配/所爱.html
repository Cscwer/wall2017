许愿墙接口文档

GET /api/ta

[ TIP ] 返回用户暗恋的人的信息

[ PARAM ] 无须参数，直接调用

比如用户 eczn 调用了该接口，后台将会返回他所爱之人

如果没有设置过，则返回 null
