许愿墙接口文档

念念不忘，必有回响

GET /api/pingpong

[ TIP ] 判断是否暗恋匹配成功，即判断用户暗恋的人是否也暗恋用户

[ PARAM ] 无须参数，直接调用即可

[ RESPONSE ] 返回值参考
