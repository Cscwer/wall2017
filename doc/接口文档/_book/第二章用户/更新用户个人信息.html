许愿墙接口文档

POST /api/user/update

根据所给字段更新用户数据

而且，更新之后，将会返回新用户数据，而且 token 也会更新，里面携带着最新的用户数据。

[ PARAM ]

user 表中的字段均可以作为参数 

http.post('/api/user/update', {
    nickname: '新名字' 
});

以上只更新用户名。
